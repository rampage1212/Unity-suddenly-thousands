
README - RETRO PIXEL IMAGE EFFECT
---------------------------------
Version: 1.0

Usage (Requires Unity Pro)
--------------------------
* Add Retro Pixel to a camera.
* Select Menu -> Component -> Image Effects -> Custom -> Retro Pixel
* Set the resolution (width and height) as you like.
* Set the number of colors and the specific color values.
* Enjoy!
